# 사진 에셋

이 폴더의 이미지는 시뮬레이션에 사용되는 **사진 기반** 스프라이트입니다.

| 파일 | 용도 |
|------|------|
| `sky.jpg` | 하늘·구름 배경 (실사) |
| `fighter.png` | 플레이어 전투기 |
| `drone.png` | 적 드론 |
| `missile.png` | 미사일 |
| `turret.png` | 발사대 |
| `explosion.png` | 폭발 효과 |

## 직접 교체하기

같은 파일명으로 **PNG/JPG**를 바꿔 넣으면 새로고침 후 반영됩니다.

- 스프라이트(전투기·드론 등): **흰 배경** 또는 단색 배경이면 자동으로 배경이 제거됩니다.
- 폭발: **검은 배경**이면 자동 제거됩니다.
- 배경: 가로로 넓은 하늘/항공 사진 권장 (1600px 이상)

## 성능

- 배경·스프라이트는 최초 로드 시 1회 처리 후 캐시됩니다.
- 용량이 크면(2MB+) 로딩이 수 초 걸릴 수 있습니다. 웹용으로 500KB~1MB 정도로 압축하는 것을 권장합니다.
